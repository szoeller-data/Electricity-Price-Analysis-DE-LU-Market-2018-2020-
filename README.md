# Electricity-Price-Analysis-DE-LU-Market-2018-2020-
An econometric analysis of hourly day-ahead electricity prices in Germany/Luxembourg, aimed to investigate the merit order effect of renewable energy generation

Project Overview
This project applies econometric methods to 17,486 hourly observations from the ENTSO-E Transparency Platform to quantify how electricity demand, solar, and wind generation affect day-ahead prices in the German/Luxembourg market.
The central question: Does renewable energy generation push electricity prices down?

Short answer: Yes, and this analysis proves it statistically.
